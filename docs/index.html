▸ loadData()

Loads all the Family data from the reducer

loadData(): Array

Type: Array

Returns
Array: returns the list of families from the backend prepares the data to fill the columns parameter defines the columns to be displayed & used by ReactTable Fields:
  • {string} code
  • {string} name family name
  • {array} familyMemberDTOList the list of family members, where we take the length to get the number of family members
  • {array} snapshotList the list of snapshots, where we take the length to get the number of snapshots
  • {object} user object of the survey taker (mentor)

▸ selectServer(username)

Will Select the backend server depending on the prefix before entered username. Username would be entered in the following formats

  • t/username -> test server
  • d/username -> demo server
  • p/username -> production server
selectServer(username: string): object
Parameters
username (string) username entered by user on frontend.
Returns
object: returns the server to be used & the username of the user
